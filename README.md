# Smart Agent Studio

Smart Agent Studio is a comprehensive platform for building, managing, and deploying intelligent AI agents and complex workflows. Built with modern web technologies and integrated with Supabase, it provides a powerful environment for both developers and non-technical users to harness the power of AI.

   ```sh

   npm install
   ```

